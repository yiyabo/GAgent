## 5. 结论与后续建模建议

### 5.1 核心发现总结

1. **类别不平衡**: 死亡组仅占 15.4%，是建模的主要挑战之一。
2. **年龄效应**: 死亡组移植时年龄略高（+3.1 岁），可能为潜在风险因子。
3. **供者指标**: 供者血红蛋白和血小板在死亡/存活组间差异较患者自身指标更为显著，提示供者健康状况对预后的重要影响。
4. **危险分层**: 具有显著的预后区分度（低危 0% → 中危 8.7% → 高危 18.2%），应作为核心协变量纳入模型。
5. **多重共线性**: 34 对特征存在强相关，需要处理以避免模型过拟合和系数不稳定。

### 5.2 对后续建模的建议

| 建议 | 理由 | 优先级 |
|------|------|--------|
| **剔除共线性特征** | 体重/体表面积、PT/INR、WBC/NEUT 等对需保留其一，降低方差膨胀因子（VIF） | 高 |
| **正则化或降维** | 使用 L1/L2 正则化或 PCA 处理剩余的中度共线性特征 | 高 |
| **SMOTE / 类别权重** | 死亡组仅 15.4%，需通过过采样或调整类别权重保证少数类学习 | 高 |
| **分层交叉验证** | 确保每折中死亡/存活比例一致，避免评估偏差 | 中 |
| **纳入供者特征** | 供者HGB和PLT的组间差异显著，是有价值的预测特征 | 中 |
| **关注危险分层交互效应** | 危险分层与其他特征的交互可能包含重要信息 | 中 |
