# 胃肠镜检查患者体验现状及舒适化护理干预研究方案
**——基于147篇最新国际文献的循证设计**

---

## 一、研究背景与立项依据

胃肠镜检查是消化道疾病诊断和治疗的重要手段，但患者在检查前、中、后常经历显著的紧张、焦虑、恐惧和躯体不适。本项目数据集已系统收集147篇最新国际文献（含26篇PMC开放获取全文、2篇PLOS ONE原文），为本研究提供了坚实的循证基础。

### 1.1 焦虑与心理负担的高发性

- **Huang等（2025）** 针对中国人群的研究显示，胃肠镜检查前焦虑发生率高达 **34.44%**，且睡眠质量（PSQI）、社会支持（SSRS）和抑郁/躯体化症状是独立预测因子。焦虑患者在SF-12各维度（GH、PF、RP、BP、RE、MH、VT、SF）得分均显著降低。
- **Zhou等（2025）** 对厦门477例无痛胃肠镜患者的研究进一步证实：担忧评分中位36分（0-75量表）；年轻患者（<40岁）是重度担忧的最强预测因子（OR最高）；农村户籍、低收入（<4000元/月）、有普通内镜不适史、粪便异常者均表现出更严重的担忧。心理韧性（PR）在社会支持（SS）与担忧之间起中介作用。
- **Albayrak等（2024）** 发现接受胃镜+结肠镜联合检查的患者焦虑水平显著高于仅行胃镜检查者，尤其是腿软/震颤、无法放松、对不良事件和死亡的恐惧。

### 1.2 患者体验测量工具的循证选择

- **Dias等（2025）** 系统评价了10项PREM量表研究，指出 **CEST** 心理测量学性能最优，**ENDOPREM** 概念覆盖最全面，但目前尚无单一工具同时满足信效度和概念完整性。所有工具均缺少"患者健康动机"维度。
- **van Bokhorst等（2025）** 在243例结肠镜患者中发现：临床医生报告的Gloucester舒适度评分（GCS）**严重低估**了患者实际经历的痛苦——中重度不适患者中92%被低估，中重度疼痛患者中90%被低估。一致性仅为κ=0.34（不适）和κ=0.47（疼痛）。**结论：必须引入患者报告结局（PREM）**。
- **Zarrin等（2025）** 验证了 **SPECS (St. Paul's Endoscopy Comfort Scale)** 在EGD中的可靠性，ICC=0.71（95%CI 0.66-0.76），适用于设施及医师水平的舒适度审计。

### 1.3 非药物干预的有效性证据

| 干预类型 | 核心证据 | 效应量 |
|----------|----------|--------|
| **虚拟现实(VR)** | Mansor等（2026）RCT, n=150, 结肠镜 | STAI降低: 29.7 vs 38.0 (Cohen's *d*=1.25); 疼痛VAS: 6.4 vs 8.6 (*d*=1.39); 满意度: 84.9 vs 63.2 (*d*=1.70) |
| **VR + 胃镜** | Turan & Özdemir（2026）RCT, n=78 | VAS-P和VAS-A显著降低, 心率/血压/呼吸率显著下降, SpO2显著升高 |
| **VR + 非镇静胃镜** | Bouman等（2026）RCT, n=89 | 未发现焦虑/疼痛客观降低，提示VR效果可能受镇静状态影响 |
| **音乐疗法** | Roelandt等（2025）Meta, 12 RCTs, n=1511 | 疼痛MD -1.02 (95%CI -1.64 to -0.41); 焦虑MD -1.07 (95%CI -1.75 to -0.39); 满意度MD 1.67 |
| **音乐疗法** | Hearn等（2026）Scoping Review, 30项研究 | 焦虑改善率71%（15/21），满意度改善率71%（10/14），疼痛改善率52%（11/21） |
| **双耳节拍音乐** | Demirci & Sezer（2024）RCT, n=96, 非镇静EGD | 状态焦虑显著降低(p=0.016), 耐受性和满意度显著升高 |
| **正念呼吸** | Sun等（2026）准实验, n=249, 经鼻内镜 | 疼痛NRS: 2.00±1.05 vs 2.67±1.46 (P<0.001); 恶心呕吐降低(P=0.01) |
| **体位护理+积极暗示** | Mu等（2026）回顾性队列, n=153, 无痛胃镜 | STAI-S: 43.23±7.68 vs 48.67±8.45 (P<0.001); GCQ显著提高; 不良反应降低 |
| **综合护理干预** | Yu & Peng（2026）RCT, n=77, 内镜息肉切除 | 炎症指标降低, 胃肠功能恢复加速, 舒适度和生活质量改善, 并发症减少 |

### 1.4 数字化宣教与延续护理

- **Fennessy等（2025）** 系统评价9项研究：数字健康干预（App、在线资源）可能提升GI内镜患者满意度，但证据质量极低、测量工具不一致。
- **Sanna等（2026）** 对355例CRC筛查患者的研究：护士主导的电话分诊服务满意度达99.2%，结肠镜参与率93.0%，肠道准备充分率94.8%。但肠道准备信息满意度仅72.4%，预约管理满意度81.6%——提示宣教环节仍需优化。
- **Liu等（2025）** 评估YouTube胃肠镜相关视频质量：整体质量不佳，LLM辅助摘要准确性尚可但可读性有限，提示专业数字化宣教材料存在缺口。
- **Bashkin等（2024）** 对191例患者的研究发现：术前信息满意度(r=0.47)和术后信息满意度(r=0.51)是患者体验的重要预测因子，信任感是核心预测变量（R²=0.61）。

### 1.5 镇静安全与患者舒适度

- **Remimazolam vs Propofol**：Su等（2026）Meta分析（14 RCTs, n=2883）显示remimazolam显著降低心肺复合不良事件（RR=0.34, 95%CI 0.27-0.44），镇静成功率相当，注射痛更少，患者满意度更高。
- **Ciprofol vs Propofol**：Zhu等（2026）Meta分析（34 trials, n=5162）显示ciprofol降低低血压（RR=0.65）、呼吸抑制（RR=0.44）、注射痛（RR=0.19）和低氧血症（RR=0.62）。
- **鼻罩氧疗**：Cheng等（2025）多中心RCT（n=1197）证实鼻罩氧疗套件显著降低镇静内镜低氧发生率（7.4% vs 12.5%, p=0.003）。
- **Esketamine辅助**：Qi等（2025）Meta分析（18 RCTs）显示esketamine联合propofol显著降低低血压（RR=0.32）和呼吸不良事件（RR=0.57），不增加心血管并发症。

## 二、研究目的

### 主要目的
1. 调查胃肠镜检查患者体验（PREM）现状及焦虑/疼痛轨迹
2. 分析社会心理及人口学影响因素
3. 评价"多模式舒适化护理干预包"对患者体验、焦虑及满意度的改善效果
4. 构建胃肠镜患者体验评价体系

### 次要目的
5. 验证"护理沟通/社会支持 → 心理韧性/焦虑降低 → 患者满意度/再次检查意愿"的中介效应模型
6. 比较普通内镜 vs 无痛内镜、胃镜 vs 肠镜 vs 联合检查的体验差异

## 三、研究设计

采用**两阶段混合研究设计**：

### 第一阶段：横断面调查
- **目的**：明确基线体验现状及影响因素
- **设计**：多中心/单中心大样本问卷调查
- **时间**：连续入组6个月

### 第二阶段：随机对照试验（RCT）
- **目的**：评价干预效果
- **设计**：平行对照，1:1随机分组
- **分组**：
  - **干预组**：多模式舒适化护理干预包
  - **对照组**：常规护理
- **盲法**：评估者盲（数据分析者不知分组）

## 四、研究对象

### 纳入标准
- 年龄 ≥ 18岁
- 行普通或无痛胃镜、肠镜、胃肠镜联合检查
- 意识清楚，能完成问卷
- 自愿参加研究并签署知情同意书

### 排除标准
- 急诊内镜
- 严重认知障碍（MMSE < 24分）
- 精神疾病（已确诊精神分裂症、双相障碍等）
- 术中发生严重并发症需抢救
- 无法完成随访

## 五、样本量计算

### 第一阶段：横断面调查
- 基于Huang（2025）和Zhou（2025）的研究，考虑15个潜在影响因素
- 按每个变量15-20例计算：15 × 20 = 300例
- 考虑20%脱落率：300 / 0.8 ≈ 375例
- **建议纳入 500 例**（确保亚组分析统计效能）

### 第二阶段：RCT干预研究
- 参考Mansor（2026）VR干预研究：STAI效应量Cohen's *d* = 1.25
- 设定双侧α=0.05，把握度1-β=0.90
- G*Power计算：每组约22例即达统计显著
- 考虑临床实际需求及多因素分析：**每组150例**
- 考虑15%脱落率：**每组150例，总计300例**

### 总样本量：约 **800 例**

## 六、测量工具与数据收集

### 6.1 推荐标准化工具组合

基于Dias等（2025）PREM系统评价、Zarrin等（2025）量表验证及Zhou等（2025）中国人群研究，推荐以下工具组合：

| 评估维度 | 推荐测量工具 | 文献依据 |
|----------|-------------|----------|
| **焦虑水平** | **GAD-7** 或 **STAI (状态-特质焦虑量表)** | Huang (2025) 使用GAD-7；Mansor (2026) 和 Mu (2026) 使用STAI |
| **疼痛** | **NRS (0-10)** | 国际通用，多项RCT采用 |
| **舒适度** | **SPECS** (St. Paul's Endoscopy Comfort Scale) | Zarrin (2025) 验证ICC=0.71 |
| **患者体验(PREM)** | **ENDOPREM** 或 **CEST** | Dias (2025): ENDOPREM概念覆盖最全，CEST心理测量学最优 |
| **社会支持** | **SSRS (社会支持评定量表)** | Zhou (2025) 证实与担忧负相关 |
| **心理韧性** | **CD-RISC (心理韧性量表)** | Zhou (2025) 证实中介效应 |
| **睡眠质量** | **PSQI (匹兹堡睡眠质量指数)** | Huang (2025) 为独立预测因子 |
| **满意度** | Likert 5点量表 + **NPS (净推荐值)** | 国际通用 |
| **再次检查意愿** | 二分类（是/否）+ Likert评分 | 多项研究采用 |

### 6.2 数据收集时间点

| 时间点 | 时间 | 收集内容 |
|--------|------|----------|
| **T0** | 检查前1周/预约时 | 人口学、基线焦虑(STAI-T)、社会支持(SSRS)、心理韧性(CD-RISC)、检查认知度 |
| **T1** | 检查当日候诊时 | 状态焦虑(STAI-S)、恐惧VAS、对麻醉/疼痛/结果的担忧、等候时间感受 |
| **T2** | 检查中/刚结束 | 疼痛NRS、恶心/腹胀评分、SPECS舒适度、隐私/沟通满意度、不良反应 |
| **T3** | 检后24h（电话/微信随访） | 总体满意度、PREM(ENDOPREM/CEST)、检后指导知晓率、再次检查意愿、NPS |
| **T4** | 检后1周（可选） | 远期满意度、筛查依从性、生活质量(SF-12) |

### 6.3 基本资料变量

| 类别 | 变量 |
|------|------|
| 人口学 | 年龄、性别、文化程度、职业、收入水平、居住地（城市/农村） |
| 检查相关 | 是否首次检查、检查类型（胃镜/肠镜/联合）、麻醉方式（普通/无痛）、陪同人员 |
| 临床因素 | 既往内镜史、腹部手术史、BMI、合并症、ASA分级 |

## 七、舒适化护理干预方案（干预组）

基于文献库中证实有效的干预措施，构建**多模式舒适化护理干预包**：

### 7.1 检查前：精准与数字化宣教
- **多媒体宣教**：图文流程手册 + 视频讲解（含肠道准备指导、麻醉注意事项、常见问题解答）
  - 参考Fennessy (2025) 数字健康干预系统评价
  - 参考Liu (2025) YouTube视频质量评估——开发专业级宣教材料
- **心理评估与疏导**：
  - 使用GAD-7快速筛查焦虑
  - 对GAD-7≥10分的高焦虑患者进行一对一心理疏导
  - 解释检查必要性、告知检查时间和可能感受、鼓励表达担忧
  - 参考Bashkin (2024): 信息满意度是体验的核心预测因子

### 7.2 候诊与术中：非药物感觉干预
- **音乐疗法**：
  - 候诊区播放舒缓背景音乐
  - 术中为患者提供耳机，播放自选音乐或双耳节拍音乐
  - 参考Demirci (2024): 双耳节拍显著降低非镇静EGD焦虑(p=0.016)
  - 参考Hearn (2026): 音乐改善焦虑率71%，满意度率71%
- **VR distraction（条件允许时）**：
  - 为普通内镜患者提供沉浸式VR放松体验
  - 参考Mansor (2026): VR在结肠镜中效应量极大(d=1.25-1.70)
  - 参考Roelandt (2025) Meta分析: VR疼痛MD -1.05
- **正念呼吸引导**：
  - 术前5分钟引导患者进行结构化正念呼吸
  - 参考Sun (2026): 疼痛NRS降低(P<0.001)，恶心减少(P=0.01)

### 7.3 术中：体位优化与舒适护理
- **体位护理**：个体化体位优化
  - 参考Mu (2026): 体位护理+积极暗示显著降低STAI-S(P<0.001)
- **积极心理暗示**：标准化话术，如"您做得很好""检查进展顺利"
- **环境优化**：
  - 分区候诊，控制等候时间
  - 提供隐私屏障，保持温度适宜（22-24°C）
- **动态观察**：
  - 持续监测生命体征、SpO2
  - 使用SPECS评估舒适度
  - 参考Cheng (2025): 鼻罩氧疗降低低氧风险

### 7.4 检查后：延续护理与闭环管理
- **即刻护理**：
  - 无痛患者苏醒质量评估
  - 检后疼痛/腹胀/恶心评估
  - 口头+书面饮食/用药/活检后注意事项
- **24h随访**（护士主导电话/微信）：
  - 参考Sanna (2026): 护士电话分诊满意度99.2%
  - 评估恢复情况，解答异常症状
  - 息肉切除后活动指导
  - 异常情况就诊提示
- **1周随访**（可选）：
  - 远期满意度评估
  - 检后健康指导知晓率
  - 再次检查意愿

## 八、统计分析计划

数据采用 SPSS 26.0 及 R 4.4.1 进行分析，检验水准 α=0.05（双侧）。

### 8.1 描述性统计
- 连续变量：均数±标准差或中位数(IQR)
- 分类变量：频数(百分比)
- 患者体验总分分布：高/中/低体验组划分

### 8.2 差异性分析
| 比较 | 方法 |
|------|------|
| 两组连续变量（干预 vs 对照） | 独立样本t检验或Mann-Whitney U |
| 多组连续变量 | 单因素ANOVA或Kruskal-Wallis |
| 分类变量 | χ²检验或Fisher精确检验 |
| 干预前后配对比较 | 配对t检验或Wilcoxon符号秩检验 |

**重点比较**：
- 首次检查 vs 非首次检查
- 普通胃肠镜 vs 无痛胃肠镜
- 胃镜 vs 肠镜 vs 胃肠镜联合
- 高文化程度 vs 低文化程度
- 等候时间短 vs 等候时间长
- 宣教充分 vs 宣教不足

### 8.3 影响因素分析
- **因变量**：患者体验总分(PREM)或总体满意度
- **方法**：多元线性回归（连续因变量）或Ordinal Logistic回归（有序分类因变量）
- **候选自变量**：年龄、性别、文化程度、是否首次检查、麻醉方式、焦虑评分、疼痛评分、等候时间、护士沟通评分、结果解释满意度、隐私保护评分、检后不适程度
- 多因素模型纳入单因素分析P<0.10的变量
- 报告标准化回归系数(β)和95%CI

### 8.4 中介效应分析（研究亮点）
- 采用Hayes的**PROCESS宏程序 (Model 4)**
- Bootstrap抽样5000次，计算95%置信区间
- **模型1**：护理沟通质量(X) → 焦虑降低(M) → 患者满意度(Y)
- **模型2**：社会支持(X) → 心理韧性(M) → 担忧水平/满意度(Y)
  - 参考Zhou (2025)的统计方法
- 报告直接效应、间接效应及中介效应占比

### 8.5 问卷信效度检验
- **信度**：Cronbach's α ≥ 0.70
- **效度**：KMO检验（≥0.60）和Bartlett球形检验（P<0.05）
- 探索性因子分析（EFA）验证结构效度

### 8.6 敏感性分析
- 意向性分析(ITT)和符合方案集(PP)分析
- 多重填补处理缺失值（缺失>5%时）

## 九、预期结果

### 9.1 横断面调查预期
1. 患者检查前焦虑发生率约30-35%（参考Huang 2025: 34.44%）
2. 首次检查者、年轻患者、农村患者焦虑水平更高
3. 医护评估的舒适度与患者自评存在显著差距（参考van Bokhorst 2025）
4. 社会支持、心理韧性、睡眠质量是体验的独立影响因素

### 9.2 干预研究预期
干预组较对照组：
| 指标 | 预期效果 | 文献依据 |
|------|----------|----------|
| 检查前焦虑评分(STAI-S) | 降低5-8分 | Mansor (2026): d=1.25 |
| 疼痛评分(NRS) | 降低1-2分 | Roelandt (2025): MD -1.02 |
| 舒适度(SPECS) | 显著提高 | Mu (2026): GCQ显著改善 |
| 总体满意度 | 提高15-20% | Mansor (2026): d=1.70 |
| 护理沟通评分 | 显著提高 | Bashkin (2024) |
| 检后指导知晓率 | 提高20-30% | Sanna (2026) |
| 再次检查意愿 | 提高10-15% | Hearn (2026) |

## 十、质量控制

1. **培训**：所有研究护士统一培训，通过一致性检验（Kappa≥0.80）
2. **随机化**：计算机生成随机序列，密封信封分配
3. **盲法**：数据分析者对分组设盲
4. **数据管理**：双人录入，逻辑核查，缺失值<5%
5. **伦理审批**：提交医院伦理委员会审批，所有参与者签署知情同意书
6. **试验注册**：在中国临床试验注册中心（ChiCTR）注册
